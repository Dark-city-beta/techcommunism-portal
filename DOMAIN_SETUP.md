# Настройка домена для GitHub Pages

Для подключения домена techcommunism.club к GitHub Pages, выполните следующие шаги:

## Настройка DNS у регистратора домена

### Вариант 1: Использование A-записей (рекомендуется)
Добавьте следующие A-записи в настройки DNS:
- 185.199.108.153
- 185.199.109.153
- 185.199.110.153
- 185.199.111.153

### Вариант 2: Использование CNAME-записи
Если ваш регистратор не позволяет использовать A-записи для корневого домена, добавьте CNAME-запись:
- @ или techcommunism.club -> Dark-city-beta.github.io

## Настройка на стороне GitHub

Файл CNAME уже добавлен в репозиторий с содержимым:
```
techcommunism.club
```

## Проверка настройки

После настройки DNS (может занять до 48 часов для полного распространения), проверьте работоспособность:

```bash
nslookup techcommunism.club
```

или

```bash
dig techcommunism.club
```

## Дополнительная информация

- GitHub Pages будет доступен по адресу: https://techcommunism.club
- Сайт будет автоматически строиться из ветки main
- Файл CNAME указывает GitHub, какой кастомный домен использовать
