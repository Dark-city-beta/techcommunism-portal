# Настройка домена для GitHub Pages

Для подключения домена к GitHub Pages, выполните следующие шаги:

## Настройка DNS у регистратора домена (techcommunism.club)

### Основной домен: techcommunism.club
#### A-записи (рекомендуется)
Добавьте следующие A-записи в настройки DNS:
- Тип: A | Имя: @ | Значение: 185.199.108.153
- Тип: A | Имя: @ | Значение: 185.199.109.153
- Тип: A | Имя: @ | Значение: 185.199.110.153
- Тип: A | Имя: @ | Значение: 185.199.111.153

#### CNAME-запись для поддомена www:
- Тип: CNAME | Имя: www | Значение: dark-city-beta.github.io

## Настройка на стороне GitHub

Файл CNAME в репозитории содержит основной домен:
```
techcommunism.club
```

## Настройка альтернативного домена (tech-communism.tk)

Для альтернативного домена tech-communism.tk настройте CNAME-запись у регистратора домена:
- Тип: CNAME | Имя: tech-communism.tk | Значение: techcommunism.club

## Проверка настройки

После настройки DNS (может занять до 48 часов для полного распространения), проверьте работоспособность:

```bash
nslookup techcommunism.club
```

или

```bash
dig techcommunism.club
```

Аналогично для второго домена:
```bash
nslookup tech-communism.tk
```

## Дополнительная информация

- GitHub Pages будет доступен по адресу: https://techcommunism.club
- Альтернативный домен https://tech-communism.tk будет перенаправлять на основной
- Сайт будет автоматически строиться из ветки main
- Файл CNAME указывает GitHub, какой кастомный домен использовать
