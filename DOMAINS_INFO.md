# Информация о доменах проекта TechCommunism Portal

## Доступные домены

Проект доступен по следующим адресам:

1. **Основной домен**: https://techcommunism.club
2. **Альтернативный домен**: https://tech-communism.tk

## Особенности использования

- Основной домен (techcommunism.club) настроен через GitHub Pages через файл CNAME
- Альтернативный домен (tech-communism.tk) должен быть настроен на уровне DNS для перенаправления на основной домен
- Оба домена ведут на одну и ту же GitHub Pages страницу

## Настройка доменов

Для информации о настройке доменов см.:
- [DOMAIN_SETUP.md](DOMAIN_SETUP.md) - общая информация о настройке

## Примечание

GitHub Pages официально поддерживает только один кастомный домен через файл CNAME. Второй домен должен быть настроен через DNS-перенаправление.
