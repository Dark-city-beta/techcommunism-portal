# Информация о доменах проекта TechCommunism Portal

## Доступные домены

Проект доступен по следующим адресам:

1. **Основной домен**: https://techcommunism.club
2. **Альтернативный домен**: https://tech-communism.tk

## Особенности использования

Оба домена ведут на одну и ту же GitHub Pages страницу. Вы можете использовать любой из них для доступа к проекту.

## Настройка доменов

Для информации о настройке доменов см.:
- [DOMAIN_SETUP.md](DOMAIN_SETUP.md) - общая информация о настройке
- [REDIRECT_SETUP.md](REDIRECT_SETUP.md) - информация о перенаправлениях между доменами

## Примечание

GitHub Pages официально поддерживает только один кастомный домен через файл CNAME, но оба наших домена настроены на уровне DNS для обслуживания одного и того же сайта.
