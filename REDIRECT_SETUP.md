# Настройка перенаправления между доменами

## Перенаправление с tech-communism.tk на techcommunism.club

Для корректного перенаправления с домена tech-communism.tk на основной домен techcommunism.club, необходимо выполнить следующие настройки:

### На стороне DNS (у регистратора tech-communism.tk)

1. Настройте CNAME-запись:
   - tech-communism.tk -> techcommunism.club

2. Или настройте A-записи на те же IP-адреса, что и для основного домена:
   - 185.199.108.153
   - 185.199.109.153
   - 185.199.110.153
   - 185.199.111.153

### Настройка GitHub Pages

GitHub Pages может обслуживать только один основной домен через файл CNAME. 
Для обслуживания обоих доменов потребуется настроить перенаправление на уровне DNS или использовать сервисы, которые поддерживают обслуживание нескольких доменов.

### Варианты реализации

1. **DNS-перенаправление** (рекомендуется):
   - Настройте DNS так, чтобы оба домена вели на один и тот же GitHub Pages сайт
   - В файле CNAME остается только основной домен (techcommunism.club)

2. **HTTP-редирект**:
   - Можно использовать JavaScript для перенаправления с одного домена на другой
   - Это менее предпочтительно с точки зрения SEO и производительности

### Рекомендуемая настройка

Для простоты и эффективности рекомендуется:
1. Оставить в файле CNAME основной домен (techcommunism.club)
2. Настроить DNS для tech-communism.tk так, чтобы он тоже указывал на GitHub Pages
3. Оба домена будут отображать один и тот же контент
